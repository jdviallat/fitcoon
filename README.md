# Fitcoon

**Fitcoon** est ton carnet d'entraînement musculation & cardio — simple, complet et **100 % gratuit**.

- Suivi des séances (séries, charge, RPE), minuteur de repos, historique
- Variations d'exécution pour chaque exercice (position, amplitude, tempo, pauses) avec « ce que ça change »
- Programmes périodisés (prise de masse, force, sèche, remise en forme, débutant)
- Bibliothèque d'exercices détaillés + illustration du mouvement
- Progrès automatiques : records, volume, calories, récupération musculaire
- Apple Watch, widget, HealthKit (optionnel), Fitbit (optionnel)
- **Tes données restent sur ton téléphone.** Sans publicité.
